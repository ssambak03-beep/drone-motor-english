# Drone Motor English

Personal mobile-first web app for practicing English expressions used in drone motor sales and technical meetings.

## Run

Open `index.html` in a browser. The first version is a static app and stores cards, progress, and records in the browser's local storage.

## First Version

- Daily 30-minute routine flow.
- Review-again cards first, then new cards.
- English-first speaking practice.
- Korean-first recall practice.
- Self-check states: memorized, uncertain, review again.
- Card add, edit, and delete.
- Simple local study records.
